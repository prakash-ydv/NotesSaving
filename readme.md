## How to Use ?

- Copy the repo to your local machine

```
git clone https://github.com/prakash-ydv/NotesSaving.git
```
- Change Current Directory

``` 
cd /NotesSaving
```
- install node module

```
npm install
```

- install dependencies

```
npm i express ejs cookie-parser nodemon
```

- run using nodemon

```
npx nodemon
```

- Redirect to localhost:3000
